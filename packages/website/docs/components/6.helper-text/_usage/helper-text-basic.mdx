### Helper Text with Icon

The `glu-helper-text` component displays an icon alongside text, useful for displaying validation messages, tooltips, or inline notifications.

#### Example Usage

<Tabs>
  <TabItem value="html" label="HTML">

    ```html live
      <div style={{ display: 'flex', flexDirection: 'column', gap: '1rem', padding: '1rem' }}>
        <glu-helper-text icon="information-circle">
          This is an informative message.
        </glu-helper-text>

        <glu-helper-text icon="check-circle" is-solid-icon="true">
          This is a success message.
        </glu-helper-text>
      </div>
    ```

  </TabItem>
  <TabItem value="jsx" label="React">
