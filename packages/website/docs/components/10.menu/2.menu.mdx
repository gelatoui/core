The `glu-menu` component is a flexible navigation component that supports multiple layouts and responsive behavior. It provides three main layout types and adapts to mobile views automatically when `is-responsive` is enabled.

---

## Basic Usage

### Example 1: Logo-Left Layout

<Tabs>
  <TabItem value="html" label="HTML">

```html live

  <img src="https://placehold.jp/120x40.png" slot="logo" alt="Logo" width="120" />
```

## Props

| Prop           | Type                              | Default       | Description                                                                 |
|----------------|-----------------------------------|---------------|-----------------------------------------------------------------------------|
| `type`         | `'logo-left'|'logo-center'|'menu-center'` | `'logo-left'` | Defines the layout type of the menu                                        |
| `logo`         | `string`                          | -             | URL for the logo image (overrides logo slot)                                |
| `is-sticky`    | `boolean`                         | `false`       | Makes the menu stick to the top of the viewport                             |
| `is-responsive`| `boolean`                         | `false`       | Enables mobile-responsive behavior                                         |

---

## Slots

| Name            | Description                                                                 |
|-----------------|-----------------------------------------------------------------------------|
| `logo`          | Content for the logo section (overridden by `logo` prop if set)             |
| `menu-items`    | Main navigation items for the menu                                         |
| `right-actions` | Additional actions or buttons displayed on the right side                  |
