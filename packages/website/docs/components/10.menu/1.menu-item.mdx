The `glu-menu-item` component represents an item in a menu, supporting different types such as labels, dropdowns, and buttons. It allows flexible content placement using slots.

---

## Basic Usage

<Tabs>
  <TabItem value="html" label="HTML">

  ```html live
  <div style={{ display: 'flex', gap: '1rem', alignItems: 'center' }}>
    <glu-menu-item type="label">
      Label Item
    </glu-menu-item>

    <glu-menu-item type="dropdown">
      <div>Dropdown Item</div>
      <div slot="dropdown-content">Dropdown Content</div>
    </glu-menu-item>

    <glu-menu-item type="button">
      <glu-button>Button Item</glu-button>
    </glu-menu-item>
  </div>
  ```
  </TabItem>

  <TabItem value="jsx" label="React">

  ```jsx live
  import { GluMenuItem, GluButton } from '@gelato-ui/react';

  function App() {
    return (
      <>
        <GluMenuItem type="label">Label Item</GluMenuItem>
        <GluMenuItem type="dropdown">
          <div slot="dropdown-trigger">Dropdown Item</div>
          <div slot="dropdown-content">Dropdown Content</div>
        </GluMenuItem>
        <GluMenuItem type="button">
          <GluButton>Button Item</GluButton>
        </GluMenuItem>
      </>
    );
  }
  ```
  </TabItem>
