Links are interactive elements that navigate users to different locations, either within the same page or to external websites. They can be customized with different styles, targets, and sizes to fit various design needs.

## Usage

### Link sizes

The `glu-link` component provides a flexible way to create hyperlinks with customizable sizes and targets.
