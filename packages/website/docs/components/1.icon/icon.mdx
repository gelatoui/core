Our icon component comes with the complete set of [Heroicons v2.0](https://heroicons.com/) in both outline and solid variants. Simply use the kebab-case name of any Heroicon:

## Usage

### Icon Variants

<Tabs>
  <TabItem value="html" label="HTML">

    ```html live
      <div style={{ display: 'flex', gap: '1rem', padding: '1rem', alignItems: 'center' }}>
        <glu-icon name="academic-cap"></glu-icon>
        <glu-icon name="academic-cap" variant="solid"></glu-icon>
      </div>
    ```

  </TabItem>
  <TabItem value="jsx" label="React">

    ```jsx live
      <div style={{ display: 'flex', gap: '1rem', padding: '1rem', alignItems: 'center' }}>
        <GluIcon name="academic-cap" />
        <GluIcon name="academic-cap" variant="solid" />
      </div>
    ```
  </TabItem>
  <TabItem value="vue" label="Vue">

    ```html
      <div style="display: flex; gap: 1rem; padding: 1rem; align-items: center;">
        <glu-icon name="academic-cap" />
        <glu-icon name="academic-cap" variant="solid" />
      </div>

      <script setup>
        import { GluIcon } from '@gelato-ui/vue';
      </script>
    ```

  </TabItem>
  <TabItem value="angular" label="Angular">

    ```html
      <div style="display: flex; gap: 1rem; padding: 1rem; align-items: center;">
        <glu-icon name="academic-cap"></glu-icon>
        <glu-icon name="academic-cap" variant="solid"></glu-icon>
      </div>
    ```

    ```ts
      import { Component } from '@angular/core';
      import { GluIcon } from '@gelato-ui/angular';

      @Component({
        standalone: true,
        imports: [GluIcon],
        template: `...` // Template content from above
      })
      export class MyComponent {}
    ```
  </TabItem>
</Tabs>

### Customization

We provide all **Heroicons** by default, including both **outline** and **solid** variants. However, you're free to use any **SVG icons** or an **icon library** of your choice.

:::note
We're actively working on a feature that will allow you to add custom SVG icons. Stay tuned for updates!
:::
