The `glu-icon` component is a versatile element that renders SVG icons from [Heroicons v2.0](https://heroicons.com/). It supports multiple visual variants, customizable sizes, and color adjustments. This makes it easy to integrate scalable icons into your projects with a consistent style.

## Basic Usage

### Example 1: Simple Icon

Display a basic icon by specifying the `name` property.

<Tabs>
  <TabItem value="html" label="HTML">

  ```html live
  <glu-icon name="arrow-up"></glu-icon>
  ```

  </TabItem>
  <TabItem value="jsx" label="React">

  ```jsx live
  <GluIcon name="arrow-up" />
  ```

  </TabItem>
  <TabItem value="vue" label="Vue">

  ```html
  <glu-icon name="arrow-up"></glu-icon>

  <script setup>
  import { GluIcon } from '@gelato-ui/vue';
  </script>
  ```

  </TabItem>
  <TabItem value="angular" label="Angular">

  ```html
  <glu-icon name="arrow-up"></glu-icon>
  ```

  ```ts
  import { Component } from '@angular/core';
  import { GluIcon } from '@gelato-ui/angular';

  @Component({
    standalone: true,
    imports: [GluIcon],
    template: `<glu-icon name="arrow-up"></glu-icon>`
  })
  export class MyComponent {}
  ```

  </TabItem>
</Tabs>

## Icon Sizes

Adjust the dimensions of the icon using the `size` property, which overrides both the `width` and `height`. Alternatively, you can set custom values for `width` and `height`.

<Tabs>
  <TabItem value="html" label="HTML">

  ```html live
  <div style={{ display: 'flex', gap: '1rem' }}>
    <div style={{ display: 'flex', gap: '1rem', alignItems: 'center' }}>
      <glu-icon name="arrow-up" size="16"></glu-icon>
      <glu-icon name="arrow-up" size="24"></glu-icon>
      <glu-icon name="arrow-up" size="32"></glu-icon>
    </div>
    <div style={{ display: 'flex', gap: '1rem', alignItems: 'center' }}>
      <glu-icon name="arrow-up" width="16" height="16"></glu-icon>
      <glu-icon name="arrow-up" width="24" height="24"></glu-icon>
      <glu-icon name="arrow-up" width="32" height="32"></glu-icon>
    </div>
  </div>
  ```

  </TabItem>
  <TabItem value="jsx" label="React">

  ```jsx live

    <GluIcon name="arrow-up" size={16} />

  ```
