# Button Component

import ButtonReadme from '@gelato-ui/components/dist/readme/components/button/readme.md'

  <glu-button target="_blank" rel="noopener noreferrer" href="https://www.figma.com/design/jSs4mJxj7dJbXsCE1IHtqt/GelatoUI-Design-System-(V-0.1.0)?node-id=239-4021&t=MItYcxbVFzFdt8Oz-0">Figma</glu-button>

The `glu-button` component is a highly customizable button element that supports various appearances, sizes, and types. It functions either as a standard button or as a link when an `href` is provided.

## Basic Usage

### Example 1: Simple Button

Use the button as a clickable element.

<Tabs>
  <TabItem value="html" label="HTML">

  ```html live
  <glu-button>Click Me</glu-button>
  ```

  </TabItem>
  <TabItem value="jsx" label="React">

  ```jsx live
  <GluButton>Click Me</GluButton>
  ```

  </TabItem>
  <TabItem value="vue" label="Vue">

  ```html
  <glu-button>Click Me</glu-button>

  <script setup>
  import { GluButton } from '@gelato-ui/vue';
  </script>
  ```

  </TabItem>
  <TabItem value="angular" label="Angular">

  ```html
  <glu-button>Click Me</glu-button>
  ```

  ```ts
  import { Component } from '@angular/core';
  import { GluButton } from '@gelato-ui/angular';

  @Component({
    standalone: true,
    imports: [GluButton],
    template: `<glu-button>Click Me</glu-button>`
  })
  export class MyComponent {}
  ```

  </TabItem>
</Tabs>

## Button in a Form

Use the button as a submission control within a form.

<Tabs>
  <TabItem value="html" label="HTML">

  ```html live
  <form onsubmit="alert('Form submitted'); return false;">
    <glu-button type="submit">Submit Form</glu-button>
  </form>
  ```

  </TabItem>
  <TabItem value="jsx" label="React">

  ```jsx live
  function FormExample() {
    const handleSubmit = (e) => {
      e.preventDefault();
      alert('Form submitted');
    }

    return (
      <form onSubmit={handleSubmit}>
        <GluButton type="submit">Submit Form</GluButton>
      </form>
    );
  }
  ```

  </TabItem>
  <TabItem value="vue" label="Vue">

  ```html
  <form @submit.prevent="() => alert('Form submitted')">
    <glu-button type="submit">Submit Form</glu-button>
  </form>

  <script setup>
  import { GluButton } from '@gelato-ui/vue';
  </script>
  ```

  </TabItem>
  <TabItem value="angular" label="Angular">

  ```html
  <form (ngSubmit)="onSubmit()">
    <glu-button type="submit">Submit Form</glu-button>
  </form>
  ```

  ```ts
  import { Component } from '@angular/core';
  import { GluButton } from '@gelato-ui/angular';

  @Component({
    standalone: true,
    imports: [GluButton],
    template: `
      <form (ngSubmit)="onSubmit()">
        <glu-button type="submit">Submit Form</glu-button>
      </form>
    `
  })
  export class MyComponent {
    onSubmit() {
      alert('Form submitted');
    }
  }
  ```

  </TabItem>
</Tabs>
