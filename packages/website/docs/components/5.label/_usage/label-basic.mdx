### Label

The `glu-label` component is a flexible label element with optional tooltip and character counter (right text) support.

#### Example Usage

<Tabs>
  <TabItem value="html" label="HTML">

    ```html live
      <div style={{ display: 'flex', flexDirection: 'column', gap: '1rem', padding: '1rem' }}>
        <glu-label>Simple Label</glu-label>

        <glu-label show-tooltip>
          Label with Tooltip
          <span slot="tooltip">This is a helpful tooltip</span>
        </glu-label>

        <glu-label show-right-text>
          Label with Counter
          <span slot="right-text">100 / 100</span>
        </glu-label>

        <glu-label show-tooltip show-right-text>
          Label with Tooltip and Counter
          <span slot="tooltip">This is a helpful tooltip</span>
          <span slot="right-text">50 / 100</span>
        </glu-label>
      </div>
    ```

  </TabItem>
