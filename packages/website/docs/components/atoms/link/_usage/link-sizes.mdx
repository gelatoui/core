### Link sizes

The `glu-link` component provides a flexible way to create hyperlinks with customizable sizes and targets.

#### Example Usage

<Tabs>
  <TabItem value="html" label="HTML">

    ```html live
      <div style={{ display: 'flex', flexDirection: 'column', gap: '1rem', padding: '1rem' }}>
        <glu-link href="#" size="small">Small Link</glu-link>
        <glu-link href="#" size="medium">Medium Link</glu-link>
        <glu-link href="#" size="large">Large Link</glu-link>
      </div>
    ```

  </TabItem>
  <TabItem value="jsx" label="React">

    ```jsx live
      <div style={{ display: 'flex', flexDirection: 'column', gap: '1rem', padding: '1rem' }}>
        <GluLink href="#" size="small">Small Link</GluLink>
        <GluLink href="#" size="medium">Medium Link</GluLink>
        <GluLink href="#" size="large">Large Link</GluLink>
      </div>
    ```

  </TabItem>
  <TabItem value="vue" label="Vue">

    ```html
      <div style="display: flex; flex-direction: column; gap: 1rem; padding: 1rem;">
    ```

  </TabItem>
