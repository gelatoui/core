# About

Welcome to the official documentation for **@gelato-ui**! This guide will help you quickly integrate and use our components to build elegant, flexible, and responsive web pages with ease.

@gelato-ui is a modern library built on **web components**, ensuring compatibility across all major browsers and frameworks. Whether you're working with React, Angular, Vue, or plain JavaScript, @gelato-ui seamlessly integrates into your workflow with full Figma support.
